## Getting started

1. Clonez le dépôt :

```bash
git clone <repository-url>
```

2. Installez les dépendances :

```bash
cd EcoTrack
npm install
```

3. Lancez le projet en mode développement :

```bash
npm run dev
```

4. Lancez le serveur nodejs :

```bash
npm run dev:server
```
